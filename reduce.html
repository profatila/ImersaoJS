Metódo reduce()

  • O método executa uma função redutora para o elemento de matriz.
  • Retorna um único valor: o resultado acumulado da função.
  • Não executa a função para elementos de matriz vazios.
  • Não altera a matriz original.
Objetivo(s):

Subtrair Números em uma matriz:

    const numeros = [175, 50, 25];

    document.getElementById("demo").innerHTML = numeros.reduce(myFunc);
