Método filter()

  • O método cria uma nova matriz preenchida com elementos que passam em um teste fornecido por uma função.
  • O método não executa a função para elementos vazios.
  • O método não altera a matriz original.
Objetivo(s):

Criar um Array apenas com os maiores de idade.

Executando Método filter() com a função de callback para verificar se é maior de idade 

    const idades = [32, 33, 16, 40];
    const resultado = idades.filter(checkAdult);

    function checkAdult(idade) {
        return idade >= 18;
    }

O Array inicial segue com os mesmo valores, apenas a variavel resultado foi filtrada: 

    idades = [32, 33, 16, 40]
    resultado = [32,33,40]
